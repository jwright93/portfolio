FitWork

Single page gym finder/review web application inspired by Yelp

Inspired by Yelp, FitWork is a web application that gives users the ability to find and review fitness activities in their area. FitWork was built on Ruby on Rails as well as React/Redux.

PackOnce

Mobile application that aims to make it easier to pack for trips

PackOnce was built using React Native. Data about past trips is stored on a PostgreSQL database accessed via a Django backend, hosted on Heroku. PackOnce allows people people make and re-make packing lists that are largely the same every time they take a vacation. A suggested item feature allows users to not have to start from scratch when they're building their packing list.

Cliffed

JavaScript game in which players are challenged to outrun the cliff for as long as possible

Cliffed is a one player game in which the objective is to avoid the top of the screen for as long as possible. Cliffed uses vanilla JavaScript and jQuery for overall structure and game logic as well as HTML5 Canvas for DOM manipulation and rendering. The player can use tactics such as wrapping around to the other side of the screen to make sure it makes it to the next branch. Once the player is cutoff by the top of the screen, they are cliffed and the game is over.
